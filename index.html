Nail Size Guide

Find your perfect fit! Common nail sizes explained.

Common nail sizes, denoted by "penny" (d), range from 2d (1 inch) to 10d (3 inches) for sizes 1-10. Generally, each 1d increase adds 1/4 inch in length, with 6d and 8d being standard for framing.

Common Nail Size Chart (Inches)

Size Length
3d 1-1/4" (1.25")
4d 1-1/2" (1.5")
5d 1-3/4" (1.75")
6d 2"
7d 2-1/4" (2.25")
8d 2-1/2" (2.5")
9d 2-3/4" (2.75")
10d 3"

💡 Tip: Use a measuring tape or ruler. If between sizes, go up!
